Geluid vormgeven voor de Dove mensen

De opdracht

Als opdracht heb ik een filmfragment van het Film blade runner vormgegeven voor mensen die doof zijn geboren. Ik moest geluid die mensen met een normaal gehoor horen, vormgeven voor dove mensen. Dat was moeglijk met CSS en Javascript. Er waren vier principes die ik moest rekening houden voor mijn werkstuk. Ten eerste moest ik een de film bekijken om het goed te begrijpen en wat de sfeer in elkaar zit. Ten tweede een storyboard schetsen gemaakt hoe ik Het geluid wilde vormgeven. Met CSS heb ik mijn ideen kunnen implementeren. Uiteindelijk kon ik de hele video vormgeving. Met deze vak kon ik meer CSS animaties maken.
